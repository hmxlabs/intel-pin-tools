# Intel Pin Tools

This repository contains additional tools for use with Intel's PIN tool. See Intel's documentation here:
https://www.intel.com/content/www/us/en/developer/articles/tool/pin-a-dynamic-binary-instrumentation-tool.html

The content of this repository should be placed in
```
<pin_root>/source/tools/
```

Each tool has its own README file with further instructions for its use.
